# PokeApiDotNet

PokeApiDotNet is a .NET client library project for accessing [PokéAPI](https://pokeapi.co/).

Currently, we have only one package available.
## [PokeApi.V2](/PokeApi.V2/README.md) [![Version](https://img.shields.io/nuget/v/PokeApi.V2?logo=nuget&style=flat-square)](https://www.nuget.org/packages/PokeApi.V2/) [![Nuget downloads](https://img.shields.io/nuget/dt/PokeApi.V2?label=nuget%20downloads&logo=nuget&style=flat-square)](https://www.nuget.org/packages/PokeApi.V2/)  
PokeApi.V2 is a .NET client library for accessing [PokéAPI API v2](https://pokeapi.co/docs/v2).

Supports in-memory caching and distributed caching (e.g. Redis).
